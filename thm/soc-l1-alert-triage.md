# THM - SOC L1 Alert Triage
**Platform:** TryHackMe

## What It Covered
How SOC analysts systematically approach and prioritize 
incoming security alerts.

## What I Learned
- Not every alert is a real threat — filtering noise 
  is a core analyst skill
- Triage follows a structured process: identify, 
  assess, prioritize, escalate
- Speed and accuracy both matter when handling live alerts
